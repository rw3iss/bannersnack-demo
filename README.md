# Bannersnack demo

# Setup:
```
cd client
npm install

cd server
npm install
```

# Start servers:
```
cd client
npm start

cd server
npm start
```

# Development:
```
cd client
npm start

cd server
npm run dev
```

# Or if you have tmux installed:
```
./tmux.sh
```
